## Abstract

Generative AI systems can produce highly plausible interpretations, yet plausibility alone does not guarantee execution admissibility. This paper presents a layered neuro-symbolic interpretation runtime that explicitly separates candidate generation from execution eligibility. The framework combines candidate proposal, typed semantic role filtering, well-formedness constraints, policy arbitration, action gating, and audit tracing.

**Key Result:** 20/20 decision-match accuracy on a calibration set spanning clear-accept, clear-reject, review-boundary, and override-mixed families.

## Core Claim

> *Plausibility and permissibility are distinct layers of competence.*
